# RickyAndMorty

Приложение написано по мотивам мультсериала "Рик и Морти". В качестве основного языка разработки взят Kotlin, также используется Java.

## Основная информация

При запуске приложения отображается картинка заставки(Splash Screen). Для этого был создан отдельный стиль темы для приложения и применен в качестве стиля по умолчанию при запуске, далее при создании главной активити тема меняется на постоянную для приложения.

Приложение содержит нижнюю навигацию (BottomNavigationView) с 3 вкладками: персонажи, локации, эпизоды. Каждая вкладка имеет доступ к фильтрации и поиску для этой вкладки. При нажатии на элемент из списка открывается экран с детальной информацией.

Приложение поддерживает кеширование и имеет возможность работать без интернета по основным вкладкам и детальным экранам. Фильтрация и поиск так же работают без интернет соединения. Все вкладки поддерживают Pull-to-Refresh. Во время загрузки данных отображается индикатор выполнения(Progress_Bar) . Для работы с картинками использовалась библиотека Glide. Для фрагментов, содержащих RecyclerView, созданы адаптеры, отображающие списки. Также для каждого фрагмента создана своя ViewModel, которая получает данные и обрабатывает изменения.

## Функционал приложения

При запуске приложения запускается Splash Screen(Экран заставки). Слева до Android 12, справо new api splashScreen Android 12.

<img src="https://github.com/Pipssec/RickyAndMorty/assets/98549196/72b7bb65-9902-4ac8-a82e-f934c427bce1" width="200" height="400">

<img src="https://github.com/Pipssec/RickyAndMorty/assets/98549196/2c414fcf-0b02-4c14-bd3f-35e72dd27395" width="200" height="400">


После загрузки Splash Screen, по дефолту у нас задан главный экран со списком персонажей. В приложении есть навигация между 3 вкладками: персонажи, локации и эпизоды. Списки создаются при помощи RecyclerView. Данные получаем с официального сайта "https://rickandmortyapi.com/", с помощью библиотеки Retrofit. Подгрузка данных идет постранично с помощью библиотеки Pagging3, и сразу же сохраняются в нашу базу данных с помощью библиотеки Room, для поддержки работы без интернета.

Вкладка список персонажей.

На данной вкладке отображается кнопка фильтра, поиска и список персонажей. При нажатии на персонажа из списка, мы переходим во фрагмент в котором показываем детальную информацию по персонажу и отдельным списком отображаем список эпизодов в котором снимался персонаж. При нажатии на эпизод, идет переход на фрагмент содержащий детальную информацию этого эпизода. Также при нажатии на локацию происхождения или местоположения, происходит переход на фрагмент с детальной информации о локации.
